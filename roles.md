# tourism-company-site
Роли пользователей:

Гость (обычный пользователь):
1. Просмотр туров (каталог).
2. Поиск туров по фильтрам (страна, цена, даты).
3. Просмотр информации о компании, контактов.

Зарегистрированный пользователь (клиент):
1. Личный кабинет.
2. Бронирование тура.
3. Просмотр своих заказов.

Администратор:
1. Управление турами (добавление, редактирование, удаление).
2. Управление пользователями и заказами.

Карта страниц:

1. Главная страница(mainpage.html)
2. Окно входа и регистрации(sign.hmtl)
3. Панель администратора(administrator.hmtl)
4. Страница регистрации(register.html)
5. Личный кабинет(account.html)
6. Каталог туров(tours.html)
7. Информация о туре(inftour.html)
8. Корзина(basket.html)
9. Страница оплаты(payment.html)
10. Подтверждение заказа(approve.html)
