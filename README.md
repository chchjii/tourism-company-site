# tourism-company-site
Роли пользователей:

Гость (обычный пользователь):
1. Просмотр туров (каталог).
2. Поиск туров по фильтрам (страна, цена, даты).
3. Просмотр информации о компании, контактов.

Зарегистрированный пользователь (клиент):
1. Личный кабинет.
2. Бронирование тура.
3. Просмотр своих заказов.

Администратор:
1. Управление турами (добавление, редактирование, удаление).
2. Управление пользователями и заказами.

Карта страниц:

1. Главная страница(mainpage.html)
2. Каталог туров(tours.html)
